## Описание программы
Expense Tracker- это консольное приложение для учета личных расходов.Данные сохраняются в JSON.

## Возможности программы:
- Позволяет добавлять,просматривать,удалять и фильтровать траты по категории или дате. 
- Использует список('List') для хранения, очередь('deque') для сортировки по дате и стек для отмены последнего действия.

### Проверка корректности ввода:
- **Сумма:** не может быть отрицательной
- **Категория:** не может быть пустой
- **Дата** должна быть в формате ГГГГ-ММ-ДД
- **Индекс** число в допустимом диапозоне
